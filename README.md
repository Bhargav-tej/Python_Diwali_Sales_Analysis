# Python_Diwali_Sales_Analysis
Description
This project focuses on analyzing Diwali sales data using Python. The goal is to clean the data, perform exploratory data analysis (EDA), extract insights, and visualize key findings through charts.

# Project Overview
The Diwali Sales Analysis project provides insights into customer behavior, sales trends, and patterns during the Diwali festival season. It uses Python for data cleaning, analysis, and visualization, enabling businesses to make data-driven decisions.

# Technologies Used
Python: For data processing and analysis.
Pandas: For data cleaning and manipulation.
Matplotlib/Seaborn: For creating visual charts.
Jupyter Notebook: For conducting EDA and presenting visuals.

# Data Cleaning
Removed null values and duplicates.
Standardized date and categorical fields.
Addressed missing data using appropriate imputation methods.
Filtered data for specific Diwali-related trends.

# Exploratory Data Analysis (EDA)
Customer Segmentation: Identified key customer demographics (age, gender, region).
Sales Trends: Analyzed sales performance during Diwali.
Product Insights: Identified the most popular product categories.
Revenue Analysis: Breakdown of revenue by location and customer group.

# Insights
Key findings from the data analysis include:

Higher sales during the first two weeks of Diwali.
Women tend to purchase more during festive sales compared to men.
Electronics and apparel are the most popular product categories.
Customers aged 25-40 contributed to the majority of the sales.
